# nhentai-downloader-web

Generated by Codex(ChatGPT)

A Flask Web UI for [doujinshi-dl](https://github.com/RicterZ/doujinshi-dl) with the [doujinshi-dl-nhentai](https://pypi.org/project/doujinshi-dl-nhentai/) plugin.
It downloads manga from nhentai.net using gallery IDs and `doujinshi-dl --token`.

![screenshot](./src/screenshot.png)

---

## Features

- Docker and local deployment
- Password-protected Web UI
- Download by nhentai gallery ID
- API token configured only through deployment environment variables
- Background download queue with task status
- Real-time download logs through the debug log viewer
- CBZ output with original download directories removed

## Docker Run

```bash
docker run -d -p 61234:61234 \
  -e NHENTAI_PASSWORD=yourpassword \
  -e DOUJINSHI_DL_TOKEN=YOUR_API_TOKEN \
  -v /your-path:/nhentai \
  1030283726/nhentai-downloader-web
```

Visit http://localhost:61234

`DOUJINSHI_DL_TOKEN` is applied by running `doujinshi-dl --token "YOUR_API_TOKEN"` before downloads. The Web UI does not accept or expose tokens.

## Local Run

```bash
git clone https://github.com/ezn24/nhentai-downloader-web
cd ./nhentai-downloader-web
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

export NHENTAI_PASSWORD=yourpassword
export DOUJINSHI_DL_TOKEN=YOUR_API_TOKEN
export DOWNLOAD_PATH=/path/to/your/dir
python nhentai.py
```

Visit http://localhost:61234

Token input is intentionally not available in the Web UI. Set `DOUJINSHI_DL_TOKEN` in the process or container environment before starting the app.

## ENV

| Variable | Description | Default | Example |
| --- | --- | --- | --- |
| `NHENTAI_PASSWORD` | Password required to access the web UI. Stored in browser cookie for 30 days. | `admin` | `mystrongpassword` |
| `DOUJINSHI_DL_TOKEN` | API token saved through `doujinshi-dl --token` before downloads. | empty | `YOUR_API_TOKEN` |
| `DOWNLOAD_PATH` | Directory where downloaded doujins will be saved inside the container. | `/nhentai` | `/download` |
| `DEFAULT_FORMAT` | Default filename format. | `%a%t` | `%a%t-%i` |
| `DOUJINSHI_DL_URL` | Mirror/base URL used by `doujinshi-dl-nhentai`. | `https://nhentai.net` | `https://nhentai.net` |
